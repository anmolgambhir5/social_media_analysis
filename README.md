Overview


The aim of this project is to analyse different aspects of Social Media addiction by executing SQL queries. This is done by uploading the dataset on SQL server,raising meaningful questions about the data and answering those questions. Through this project, I analyze and publish observations and findings of the data.

This repository contains two files:

SOCIAL_MEDIA_ATTENTION.sql - This contains the dataset, and questions and queries on the same
SOCIAL_MEDIA_ANALYSIS_FINDINGS.docx - This contains the inferences and findings of the dataset
